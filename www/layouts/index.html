Frequently Asked Questions

Hyaline uses a "bring your own" model for LLM providers and GitHub Actions for several reasons:

  • Cost Control - You have complete visibility and control over your LLM and GitHub Actions usage. You can monitor costs directly, set spending limits, and optimize usage based on your needs.
  • Data Privacy - This approach minimizes the data that flows through Hyaline's systems. Your code remains private in your GitHub repositories and is processed directly by your chosen LLM provider. Note: The only exception is webhook payloads sent to the Hyaline-hosted (it is possible to self-host) GitHub App, which include code diffs that we ignore. Learn more about our data handling practices on our security page.
  • Simplicity - Many organizations already have enterprise agreements with LLM providers (like OpenAI, Anthropic, or Azure) and GitHub. You can leverage these existing relationships, compliance approvals, and negotiated rates rather than managing another vendor relationship.

This approach is also why we are able to offer organization-wide licensing rather than per-user or usage-based pricing. It's also important to note that GitHub Actions and several LLM providers offer a free tier that may be sufficient for your needs. See our documentation about supported LLMs to learn more.

You can use Hyaline without a license for evaluation and demonstration purposes only. For all other uses, a license is required. We offer free licenses for open source projects, non-profit organizations, and educational institutions—just reach out to request one. All other companies need to purchase a license for continued use beyond evaluation.

To make your experience with Hyaline as seamless as possible. We don't want you to have to spend time hunting for license files, dealing with activation issues, or managing renewals across your team. We trust you to purchase appropriate licenses, and in return, you get to focus on what matters—making your documentation useful.

While Hyaline is source-available (you can view the code), we've chosen a proprietary license model to ensure we can sustainably develop and support the product long-term. This approach allows us to dedicate resources to making Hyaline the best documentation tool possible while maintaining fair pricing and offering free licenses to qualifying organizations.
